## Парсер характеристик авто с auto-data.net

Скрипт загружает страницу автомобиля с [auto-data.net (рус)](https://www.auto-data.net/ru/) и выводит основные характеристики.

### Установка

```bash
pip install -r requirements.txt
```

### Запуск

1. Открой в браузере [https://www.auto-data.net/ru/](https://www.auto-data.net/ru/), найди нужную модель и открой страницу **конкретной модификации**.
2. Скопируй URL (например: `https://www.auto-data.net/ru/skoda-octavia-iii-scout-1.6-tdi-110hp-4x4-56218`).
3. В терминале из папки проекта:

```bash
python parse_car.py
```

4. Вставь ссылку по запросу скрипта.

### Что выводится

- **Марка**
- **Модель**
- **Модификация (двигатель)**
- **Начало выпуска – Окончание выпуска**
- **Крутящий момент (Нм)**
- **Мощность (л.с.)**
- **Вес (кг)**
- **Разгон 0–100 км/ч**
- **Максимальная скорость**

Если запрос к сайту не удаётся (например, блокировка), скрипт предложит сохранить страницу в папку `cars/` или вставить HTML из буфера обмена.
